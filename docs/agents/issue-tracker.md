# Issue Tracker

**Type:** GitHub Issues
**Repo:** cocchialorenzo9/vibe-marathon
**CLI:** `gh` (GitHub CLI)

## How skills interact with issues

- List open issues: `gh issue list`
- Create an issue: `gh issue create --title "..." --body "..."`
- View an issue: `gh issue view <number>`
- Add a label: `gh issue edit <number> --add-label "<label>"`
- Close an issue: `gh issue close <number>`

## PRs as triage surface

Disabled. Only GitHub Issues are triaged; pull requests are not pulled into the triage queue.
